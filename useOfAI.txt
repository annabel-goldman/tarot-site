How can I write a css media query for a 400% zoomed in screen? 
Response:
CSS media queries don't directly support zoom levels (like "400% zoom"), because zoom is typically a browser-controlled feature and doesn't change the actual dimensions of the viewport as reported to CSS‚Äîjust how it's rendered to the user. However, you can simulate a "zoomed-in" experience by targeting very small viewport widths, since zooming in reduces the effective viewport size.

Approximation Technique
To approximate a 400% zoom level, you can assume that the viewport behaves as if it's 1/4 the original size. So if your base design is for a 1440px width screen, then 400% zoom would behave roughly like a screen that's ~360px wide.

  /* Styles that apply when zoomed to ~400% */
  body {
    font-size: 2rem;
    background-color: lightyellow;
  }

    padding: 1rem;
  }
}
